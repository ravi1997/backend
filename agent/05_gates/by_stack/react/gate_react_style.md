# React Gate: Style

## Purpose

Component and code styling.

## Rules

1. **Linter**: `eslint-plugin-react`.
2. **Hooks**: `eslint-plugin-react-hooks` (rules of hooks) MUST pass.
3. **JSX**: PascalCase for components.

## Check Command

```bash
npm run lint
```

## Failure criteria

- Lint errors.
- React Hooks rule violations.
